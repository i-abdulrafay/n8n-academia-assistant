# n8n-academia-assistant
Automated Academic Administration Agent built with n8n, Advanced AI RAG, and Groq.

## How to Import This Workflow into n8n

1. Download the `academia_assistant_workflow.json` file from this repository.
2. Open your local n8n instance canvas interface.
3. Create a brand new, empty workflow.
4. Click on the canvas grid and press **Ctrl + O** (or go to the top-right menu and select **Import from File**).
5. Choose the downloaded JSON file.
6. Connect your personal Gmail and Google Sheets accounts to the newly imported credential nodes, and you're ready to run!
